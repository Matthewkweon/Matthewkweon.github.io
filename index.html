My Projects

Gym Classification

Github link: Github

This is a personal passion project that stems from my passion in Machine Learning, web development, API usage and the Gym.

This project is targetted at users who are more unfamiliar with the gym, and to make the gym experience a more inclusive space, I create a program where users can input an image of any gym equipment and the program will output a short discription of the machine, how to use it, and also link a suitable youtube tutorial video they can watch. This is designed to be easy, simple, and user friendly for all users in order to make the gym space a more judgement-free zone, as everyone knows how intimidating the gym can be for beginners.

I developed a full-stack web application using Python and JavaScript to classify gym equipment from images, leveraging OpenAI's GPT-4 Vision API. The application is web-based and phone-accessible, allowing users to upload images of unfamiliar gym machines for instant identification. This feature is particularly designed to assist older gym-goers and newcomers in navigating and effectively using gym equipment by providing usage instructions and linked tutorial videos. Throughout the project, I integrated multiple APIs, including OpenAI and YouTube, and implemented image processing techniques using Pillow, demonstrating my practical machine learning and computer vision skills. Additionally, I deployed and managed a production-ready application on Heroku, incorporating error handling, logging, and security practices such as CORS management. This project enhanced my understanding of full-stack development, API integration, and deployment strategies, while also honing my skills in machine learning, computer vision, and creating user-friendly interfaces.

I am currently in the works of expanding this outreach and after further algorithm optimization, I plan to deploy this at my local gym and paste a QR code that will allow users to access this program. Wish me luck!

Productivity Program

Github link: Github

This is a personal project stemming from my passion in productivity videos and books. Throughout my college life, I have been engrossed in the world of productivity and self-improvement, and I have always wanted to create a program that would help me focus when deep work was needed. This project was meant to help myself focus, but also can be used by others with basic knowledge of python, javascript and web-development. I am currently working on making this program into a desktop application, so that it has more control over the user's computer and can be used more effectively.

I developed a productivity tracker utilizing HTML/CSS, Python, and JavaScript, featuring a Pomodoro timer and goal-setting functionalities aimed at helping users with productivity issues. The application includes real-time activity monitoring implemented using Flask and is deployed on the Heroku infrastructure, ensuring seamless, real-time, and accessible productivity tracking. I collaborated with Pushover APIs and experimented with Twilio for notifications, integrating a responsive design that enhances user interaction and engagement. Through this project, I learned to effectively combine front-end and back-end technologies to create a cohesive and user-friendly application, while also gaining experience in deploying and managing applications on cloud platforms. The use of Pushover requires users to have the app installed for notifications, adding an extra layer of real-time communication to the productivity tracker.
