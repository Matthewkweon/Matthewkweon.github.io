Gym Classification

Github link: Github

Program link: website

This is a personal passion project that stems from my passion in Machine Learning, Web Development, API usage and the Gym.

This project is targetted at users who are more unfamiliar with the gym, and to make the gym experience a more inclusive space, I create a program where users can input an image of any gym equipment and the program will output a short discription of the machine, how to use it, and also link a suitable youtube tutorial video they can watch. This is designed to be easy, simple, and user friendly for all users in order to make the gym space a more judgement-free zone, as everyone knows how intimidating the gym can be for beginners.

I developed a full-stack web application using Python and JavaScript to classify gym equipment from images, leveraging OpenAI's GPT-4 Vision API. The application is web-based and phone-accessible, allowing users to upload images of unfamiliar gym machines for instant identification. This feature is particularly designed to assist older gym-goers and newcomers in navigating and effectively using gym equipment by providing usage instructions and linked tutorial videos. Throughout the project, I integrated multiple APIs, including OpenAI and YouTube, and implemented image processing techniques using Pillow, demonstrating my practical machine learning and computer vision skills. This project enhanced my understanding of full-stack development, API integration, and deployment strategies, while also honing my skills in machine learning, computer vision, and creating user-friendly interfaces.

I am currently in the works of expanding this outreach and after further algorithm optimization, I plan to deploy this at my local gym and paste a QR code that will allow users to access this program. Wish me luck!

Jumpknight: Android Studios App Development

Github link: Github

Youtube link: Github

JumpKnight is an engaging and challenging platform jumper game where players take on the role of a knight striving to reach the highest platform. Users control the knight's jumps using intuitive tilt-based controls and screen press duration, which dictate the direction and height of each jump. The game starts with a simple instruction screen that guides players on how to play. Once in the game, players must strategically time and aim their jumps to ascend from platform to platform. The game’s physics engine adds a layer of complexity, making each jump feel realistic and requiring precision. If a player misses a platform, they might fall to a lower level or even to the beginning, adding a thrilling element of risk and reward. Fun interactions include the ability to see the knight’s animated movements and the dynamic background that changes as the player progresses. Reaching the top platform and touching the flag triggers a celebratory end screen, providing a sense of accomplishment. JumpKnight’s blend of strategic gameplay, intuitive controls, and rewarding challenges creates an addictive and enjoyable gaming experience.

I collaborated with four other engineers on a comprehensive full-stack Android game project, JumpKnight, utilizing Android Studio, C++, Git, and Java. My primary focus was spearheading the UI/UX and front-end development to enhance the user experience for over 150 users. I designed and implemented the start and end screens, provided users with clear instructions, and ensured smooth page transitions between different game screens, contributing to an intuitive and engaging user interface. In addition to UI/UX design, I co-developed the app's front end by implementing features such as tilt-based controls and touch-jump mechanics, which significantly enhanced gameplay by allowing users to control the knight’s jumps through device orientation and screen press duration. I led the testing efforts, conducting extensive debugging to identify and resolve critical bugs such as issues with platform clipping and terminal velocity. This process involved iterative testing and optimization to improve game stability and performance across various Android devices. By coordinating closely with team members, I integrated key game components, including collision detection, physics calculations, and level design, ensuring a cohesive and functional final product. My contributions were essential in delivering a polished and enjoyable gaming experience for all users.
